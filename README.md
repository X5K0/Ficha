# ¿Qué es Ficha?
<table>
<tr>
<td>
Ficha es una herramienta ideal para generar cadenas de filtros en PHP y obtener un RCE sin subir un archivo.<br>
Jugando con el encoding podemos añadirle caracteres a una cadena en Base64 para que así el código PHP nos lo interprete.
</td>
</tr>
</table>


## Uso

```python
$ python3 ficha.py --help             
usage: ficha.py [-h] [--c C] [--rawbase64 RAWBASE64]

Generador de filtros de PHP.

options:
  -h, --help            show this help message and exit
  --c C                 Contenido que desea generar.
  --rawbase64 RAWBASE64
                        El valor base64 que desea probar, la cadena será impresa como base64 por PHP.
```

## Parámetros

- --c C:  Cadena PHP que desea inyectar.
